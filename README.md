스타벅스 공식 홈페이지의 로그인 페이지를 분석하고, 깔끔한 UI와 사용자 경험(UX)을 그대로 재현한 클론 코딩 프로젝트입니다.

🎯 Project Purpose (기획 의도)
UI 정밀 구현: 스타벅스 특유의 정갈한 레이아웃과 일관된 브랜드 컬러(Starbucks Green)를 코드로 구현하는 연습을 했습니다.

사용자 경험(UX) 최적화: 실제 로그인 시 발생할 수 있는 입력값 검증과 반응형 디자인을 고려하여 제작했습니다.

✨ Key Features (주요 기능)
반응형 레이아웃: 데스크탑뿐만 아니라 모바일 사파리(Safari) 브라우저에서도 깨지지 않는 뷰포트 최적화를 적용했습니다.

인풋 포커스 효과: 아이디와 비밀번호 입력창 클릭 시 테두리 색상이 부드럽게 변하는 인터렉션을 구현했습니다.

보안 안내 문구: 실제 사이트와 동일하게 주의사항 및 보안 안내 텍스트를 배치하여 신뢰감을 주는 UI를 구성했습니다.

🛠 Tech Stack (사용 도구)
Design: Figma (컴포넌트 및 가이드 분석)

Frontend: HTML5, CSS3

Tools: GitHub Desktop (버전 관리), Visual Studio Code
